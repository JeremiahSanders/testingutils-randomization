# StringRandomizationSourceExtensions.RandomStringLatin method

Generates a random String using ASCII Latin characters.

```csharp
public static string RandomStringLatin(this IRandomizationSource randomizationSource, int length, 
    bool uppercase = false, bool alphanumeric = false)
```

| parameter | description |
| --- | --- |
| randomizationSource | A [`IRandomizationSource`](../IRandomizationSource.md) providing values. |
| length | A string length. |
| uppercase | A flag indicating whether generated characters should be uppercase. False indicates lowercase. |
| alphanumeric | A flag indicating whether generate characters should be alphanumeric. False indicates alphabetic. |

## Return Value

A randomly-generated String of *length* characters.

## Remarks

This method is equivalent to [`RandomString`](./RandomString.md) , but uses predefined char ranges of `a` to `z` for Latin lowercase, `A` to `Z`, for Latin uppercase, and `0` to `9` for numeric.

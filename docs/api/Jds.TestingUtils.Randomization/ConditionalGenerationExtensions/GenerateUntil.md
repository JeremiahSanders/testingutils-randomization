# ConditionalGenerationExtensions.GenerateUntil&lt;TResult&gt; method

Continuously generates values using the provided generator function until a value satisfies the given condition.

```csharp
public static TResult GenerateUntil<TResult>(this IRandomizationSource randomizationSource, 
    Func<IRandomizationSource, TResult> generator, Func<TResult, bool> condition)
```

| parameter | description |
| --- | --- |
| TResult | The type of the values being generated. |
| randomizationSource | The source of randomization. |
| generator | A function that generates values of type *TResult*. This function receives this [`IRandomizationSource`](../IRandomizationSource.md) as a parameter. |
| condition | A predicate function that determines whether the generated value satisfies a condition. The first value which satisfies the condition is returned. |

## Return Value

A value of type *TResult* that satisfies the given condition.

## Remarks

This method is useful for generating values that meet specific criteria in situations where it is more efficient to simply retry value generation than to create a custom generation function.

For example, generating a pseudorandom number which is a multiple of 3 can be done as follows:

```csharp
int multipleOfThree = Randomizer.Shared.GenerateUntil(static source => source.IntPositive(), static value => value > 0 && value % 3 == 0);
```

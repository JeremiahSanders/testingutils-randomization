Creates a function which generates String Markov Chains.

```csharp
    this IRandomizationSource randomizationSource, IEnumerable<string> sources, int chainLength = 1)
```

| parameter | description |
| --- | --- |
| randomizationSource | A [`IRandomizationSource`](../IRandomizationSource.md) providing values. |
| sources | Example strings which will train the function. |
| chainLength | Desired length of Char sequences, obtained from *sources*, which will train the 'next item' selections made by the created function. |

## Return Value

A function which accepts a maximum length and returns a string, derived from the Char sequences in each String in *sources*.

## Exceptions

| exception | condition |
| --- | --- |
| ArgumentOutOfRangeException | Thrown when *chainLength* is less than 1. |
| ArgumentException | Thrown when *sources* is empty, if any items in *sources* are empty, or if *chainLength* exceeds the longest sequence in *sources*. |
